# StreamWeaver

## Video Processing Platform Dashboard
StreamWeaver is a React-based frontend dashboard that simulates a professional video processing platform. It provides interfaces for video format management, subtitle handling, audio issue tracking, copyright compliance verification, media quality analysis, delivery network monitoring, and infrastructure health visualization.

## Project Description
StreamWeaver provides a centralized workspace for managing various stages of video processing. The dashboard is designed with a modern black and beige theme inspired by professional media platforms and focuses on usability, responsiveness, and clean UI design.

### Processing Workspace
- Format Rulebook
- Search Video Formats

- Infrastructure Health Monitoring
- Global Delivery Network
- Media Quality Analyzer
- Error Recovery System

### Dashboard Statistics
- Active Jobs
- Audio Events
- Subtitle Count
- Compliance Percentage

### User Interface
- Responsive Layout

## Installation
Clone the repository:

bash
git clone https://github.com/saudamini-stackk/streamweaver-react.git

Move into the project folder:
bash
cd streamweaver-react

Install dependencies:
bash
npm install

Start development server:
bash
npm run dev
